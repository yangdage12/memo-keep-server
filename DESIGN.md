# DESIGN.md - 重要事件记录与提醒 App

## 产品名称
MemoKeep - 重要事件记录助手

## 气质与意象
清晨书桌上摊开的皮质笔记本，旁边放着一杯温热的咖啡。阳光透过百叶窗在纸面上投下规律的条纹，每一条待办事项都被用不同颜色的荧光笔标记。这是一个关于"掌控感"的产品——让混乱的生活变得井然有序。

## 视觉策略
- 图标系统：使用 Feather 图标集，线条简洁统一
- 卡片式布局：每个事件是一张独立的卡片，通过颜色标签区分分类
- 时间线视觉：首页以时间线形式展示即将到来的事件

## 配色方案
基于柔和卡片风（Soft Card UI），调整为本应用的专属色彩：

| 用途 | 色值 | 意象来源 |
|------|------|----------|
| 主色 | #6C63FF（薰衣草紫） | 专注与智慧 |
| 辅色 | #FF6584（珊瑚粉） | 温暖提醒 |
| 背景 | #F0F0F3（暖灰白） | 纸张质感 |
| 工作标签 | #6C63FF（紫色） | 专业沉稳 |
| 生活标签 | #00B894（薄荷绿） | 轻松自然 |
| 家庭标签 | #FDCB6E（暖黄） | 温馨家庭 |
| 高优先级 | #FF6B6B（柔红） | 紧急醒目 |
| 中优先级 | #FDCB6E（暖黄） | 需要关注 |
| 低优先级 | #00B894（薄荷绿） | 从容处理 |

## 字体排版
- 大标题：28px, fontWeight 800
- 卡片标题：16px, fontWeight 700
- 正文：14px, fontWeight 400
- 辅助文字：12px, color #636E72

## 动效与交互
- 卡片按压：scale 0.97, 100ms
- 新增事件：从底部滑入 Modal
- 完成事件：轻微弹跳动画 + 勾选效果

## 设计禁忌
- 不使用纯白卡片背景（保持 #F0F0F3 底色）
- 不使用硬边框分隔层级
- 不使用冷灰色文字（用暖灰 #636E72）
- 不使用矩形选中态（用颜色+小圆点）
